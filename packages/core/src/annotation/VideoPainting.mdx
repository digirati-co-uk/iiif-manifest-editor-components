# Video Annotation with Painting Motivation

Video url and a thumbnail url can be specified. 

## Requirements

- [ ] works with web video
- [ ] optional: YouTube embed
- [ ] choices for multipe source types and resolutions.
- [ ] thumbnail generator/uploader if it hasn't been specified
- [ ] ability to trim the videos (start, end - not the resolution)
