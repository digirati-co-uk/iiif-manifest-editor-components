# Configuration Reducer

This reducer allows to runtime overriders for some of the basic configurations:

- [ ] global id generator url prefix
- [ ] global id generator function
- [ ] default language
- [ ] component defaults

Use cases:
- Preferences Pages.
- Language Selectors
- When projects require multiple manifest templates, it can be used in a project type selector.
